# WP Packages Test

A WordPress plugin demonstrating new build system.

## Getting Started

### Prerequisites
- Node.js (v20 or higher recommended)
- npm

### Installation & Build

```bash
# Install dependencies
npm install

# Build the plugin
npm run build
```

The compiled assets will be generated and ready for use.

## Development

For development with hot-reload:

```bash
npm run dev
```
